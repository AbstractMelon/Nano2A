# Nano2A

An open-source micro AFHDS2A (FlySky) receiver.

## Features

- **Protocol:** AFHDS2A (FlySky)
- **Outputs:** PWM (3ch)
- **Input voltage:** 6.4–22.2V (2S–6S LiPo)
- **Weight:** < 3g
- **Size:** 13.75×15mm PCB
- **Range:** 300m+ (open field)
- **Telemetry:** Battery voltage sensing (ADC)
- **Cost:** ~$2.9 BOM + ~$0.4 PCB (~$3.30/unit)
